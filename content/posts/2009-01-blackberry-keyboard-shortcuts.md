Keyboard shortcuts are always good to know. Here is a list of the handy ones for a Blackberry 8900.<br />
<br />
<span style="font-weight: bold;">General functions</span><br />
<br />
<span style="font-style: italic;">Any text fields</span><br />
Shift+roll ball: highlight text<br />
Shift+Del: cut<br />
Alt+click ball: copy<br />
Shift+click ball: paste<br />
<br />
<span style="font-style: italic;">In lists and web pages</span><br />
Space: page down<br />
Shift+Space: page up<br />
<br />
<span style="font-weight: bold;">Home screen</span><br />
Tying: will start a search of your contacts, except the number keys that will also enter that number<br />
Green handset: recent calls. also when on a call after you went to the menus this takes you back to the <span class="blsp-spelling-error" id="SPELLING_ERROR_0">phonecall</span> screen.<br />
Q: hold down to turn on Vibrate only profile.<br />
<br />
<span style="font-weight: bold;">Messages</span><br />
E: next message with an error.<br />
R: starts a reply to the selected message, using the same address it was sent to.<br />
T: <span class="blsp-spelling-error" id="SPELLING_ERROR_1">goto</span> the top of the list.<br />
U: next unread message.<br />
P: previous day<br />
S: starts a search<br />
F: froward selected message<br />
J: next message with same subject<br />
K: prev message with same subject<br />
L: reply-to-all for selected message<br />
Del: deletes message<br />
C: compose new message<br />
V: see saved messages<br />
B: <span class="blsp-spelling-error" id="SPELLING_ERROR_2">goto</span> the bottom of the list<br />
N: <span class="blsp-spelling-error" id="SPELLING_ERROR_3">goto</span> next day<br />
<br />
Alt+I: show incoming messages<br />
Alt+O: show outgoing messages and calls<br />
Alt+P: show all phone calls, incoming and outgoing<br />
Alt+S: show <span class="blsp-spelling-error" id="SPELLING_ERROR_4">SMS</span> messages<br />
Alt+V: show voicemail messages<br />
Alt+M: show <span class="blsp-spelling-error" id="SPELLING_ERROR_5">MMS</span> messages<br />
Alt+Enter: another way to compose a message<br />
<br />
<span style="font-weight: bold;"><span class="blsp-spelling-corrected" id="SPELLING_ERROR_6">Calendar</span></span><br />
W: week view<br />
A: agenda view<br />
M: month view<br />
D: day view (keyboard shortcuts <span class="blsp-spelling-corrected" id="SPELLING_ERROR_7">don't</span> work on Day view if you have 'Enable Quick Entry=On'. I turned mine off as its so fast to create a new appointment by pressing enter)<br />
G: <span class="blsp-spelling-error" id="SPELLING_ERROR_8">goto</span> date<br />
T: goto today<br />
C: compose new <span class="blsp-spelling-corrected" id="SPELLING_ERROR_9">appointment</span><br />
Enter: compose new appointment<br />
N: next day/month<br />
P: previous day/month<br />
<br />
<span style="font-weight: bold;">Browser</span><br />
R: reload<br />
Y: history<br />
U: hide/show top status bar<br />
I: zoom in<br />
O: zoom out<br />
P: page info, possible to copy or email the URL<br />
A: add bookmark<br />
S: settings<br />
D: close browser<br />
F: find on page<br />
G: goto ULR, also shows web search, bookmarks and recent history<br />
H: goto home page<br />
J: refresh page wirh java enabled<br />
K: show bookmarks<br />
C: connecton info<br />
V: search for on page/find next on page<br />
T: top<br />
B: bottom<br />
N: forward
