PubMed updated their <a href="http://www.ncbi.nlm.nih.gov/pubmed/">main page</a> the otherday, which is nice.<br /><br /><br /><br />However there was a bug or problem with my Firefox keyword searches as they no longer worked. After a bit of digging I found <a href="http://shelved.blogspot.com/2008/04/pubmed-sends-out-few-new-blooms.html">a few handy</a> to know URLs for use with pubmed/NCBI, such as:<br /><br /><span style="font-style:italic;">PMID links</span><br /><a href="http://pubmed.com/19621437">http://pubmed.com/19621437</a><br /><br />which will open the page for the article "Generation of a conditional CREB Ser133Ala knockin mouse" PMID:19621437<br /><br /><span style="font-style:italic;">Search URL</span><br /><a href="http://www.ncbi.nlm.nih.gov/pubmed?term=nur77+dundee">http://www.ncbi.nlm.nih.gov/pubmed?term=nur77+dundee</a><br />will search PubMed with the following term "nur77 dundee"<br /><br />In any case, here is how I used the above URL to fix my Firefox keyword search bookmarks. This might be useful for other sites where right-clicking and creating a keyword bookmark doesn't work, as long as you know the correct search URL for the site in question. <br /><br />You can edit the bookmark HTML by exporting your bookmarks, making changes to the file, then deleting all the bookmarks in firefox (except the toolbar ones) and re-importing the file. No doubt there is a way to directly edit the live copy of the file but I don't know where that is saved...<br /><br />Find the entry for your PubMed keyword and remove the <pre>post_data".."</pre> part. Also update the URL to the following <pre>http://www.ncbi.nlm.nih.gov/pubmed?term=%s</pre><br />the end result will look something like this, if your keyword is set as "pub"<br /><pre>&lt;DT&gt;&lt;a href=" http://www.ncbi.nlm.nih.gov/pubmed?term=%s" SHORTCUTURL="pub" LAST_CHARSET="UTF-8"&gt;PubMed&lt;/a&gt;</pre>
