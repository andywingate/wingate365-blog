Problem: Mount an ISO image file<br /><ul><li>Good old terminal<br /></li></ul>To make a mount point "iso" and mount the CD image "image.iso"<br /><pre>$ sudo mkdir /media/iso &amp;&amp; sudo mount -o loop image.iso</pre>To unmount the ISO and clean up the /media directory<br /><pre>$ sudo umount /media/iso &amp;&amp; sudo rmdir /media/iso</pre>Always good to know the terminal commands, thanks <a href="http://www.cyberciti.biz/tips/how-to-mount-iso-image-under-linux.html">nixCraft</a>.<br /><hr /><ul><li>Add a script to nautilus</li></ul><pre>$ cd ~/.gnome2/nautilus-scripts &amp;&amp; sudo gedit mount</pre>Add this text to make the 'mount' script<br /><pre>#!/bin/bash<br />#<br /># nautilus-mount-iso<br />gksudo -u root -k /bin/echo "got r00t?"<br />sudo mkdir /media/"$*"<br />if sudo mount -o loop -t iso9660 "$*" /media/"$*"<br />then<br />if zenity --question --title "ISO Mounter" --text "$* Successfully Mounted.<br />Open Volume?"<br />then<br />  nautilus /media/"$*" --no-desktop<br />fi<br />exit 0<br />else<br />sudo rmdir /media/"$*"<br />zenity --error --title "ISO Mounter" --text "Cannot mount $*!"<br />exit 1<br />fi</pre>Now make the umount script<br /><pre>$ sudo gedit umount</pre>and here is the code<br /><pre>#!/bin/bash<br />#<br />for I in "$*"<br />do<br />foo=`gksudo -u root -k -m "enter your password for root terminal<br />access" /bin/echo "got r00t?"`<br />sudo umount "$I" &amp;&amp; zenity --info --text "Successfully unmounted /media/$I/" &amp;&amp; sudo rmdir "/media/$I/"<br />done<br />done<br />exit0</pre>Set the files as executable<br /><pre>$ chmod 755 mount umount<br /></pre>This should have added the two new scripts to the context menu. Thanks to <a class="bigusername" href="http://ubuntuforums.org/member.php?u=15731">animacide</a> @ <a href="http://ubuntuforums.org/showthread.php?t=87369">ubuntu forums</a>.<br /><hr /><ul><li>Install gMount</li></ul><pre>$ sudo apt-get install gmountiso</pre>While it's good to know the other methods this one is by far the easiest to use.
