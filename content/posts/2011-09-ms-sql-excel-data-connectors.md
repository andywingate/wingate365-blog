Data Connectors are a very handy way to give office staff access to the data they need from an MS-SQL database without any knowledge of SQL.<br />
<br />
Where central data is held in MS-SQL databases but the application level is unable to provide all the various summary and analysis functions that fast moving businesses need.<br />
<br />
Either create views and link the connectors to these or simply link the connectors to SQL statements with a read-only access to the database.
<br />
Once you have your new data connector saved it's easy to directly edit the .odc files to change the initial catalog, the view that is being accessed and user names etc.<br />

The .odc file can also be changed between accessing tables or views or SQL statements. Via 'Connection Properties' dialog found in the Data tab<br />
<br />
The main advantage of data-connector driven excel models is that the data is refreshed from the primary source either automatically or at the press of a button.<br />
<br />
Provides&nbsp;as-good-as online access&nbsp;for business&nbsp;analysis or performance&nbsp;monitoring&nbsp;projects with virtually nil&nbsp;development&nbsp;overhead.
