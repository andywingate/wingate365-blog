The **Dynamics 365 Business Central WhatsApp community** collects BC-related RSS feeds
and other news sources from across the web, all in one place. A bot posts the latest
blogs, videos, docs and repo updates so you can keep on top of what's going on in the
BC world without hunting around.

## Want to join?

the same form to submit a blog to the feed.

## What's it for?

The main **Announcements** channel is where the BC Blogs Bot posts all the new blogs
and news to help you keep on top of what is going on:

- Bloggers, YouTube channels and Microsoft Learn page updates
- LinkedIn newsletters
- GitHub repos
- Over 100 carefully selected sources

If you know a good source of BC news that doesn't show in the feed, or you're starting a
new BC blog, please give me a shout and I'll add the source.

## Groups in the community

- **Announcements** - the main channel where the collected BC blog / LinkedIn / YouTube feed is posted
- **MS Learn Updates** - updates to Learn docs (there are often a lot!), split out for that reason
- **Recent Commits** - recent changes to a curated list of BC-related GitHub repos (shoutout to [Brad Prendergast](https://www.linkedin.com/in/brad-prendergast/) for the idea and for kindly sharing his curated list of repos to watch)

## Privacy

The Announcements channel has privacy mode turned on, so your number is only shared with
the community admin.
